# Currency Exchange Frontend (React + Vite + Tailwind CSS)

This is the frontend application for the Currency Converter project.
It is built using React, Vite, and Tailwind CSS and consumes a backend
API deployed on Microsoft Azure to convert currencies in real time.

---

## Features

- Clean and responsive UI
- Currency selection with country flags
- Real-time currency conversion
- Integrated with backend REST API
- Fully responsive design (mobile, tablet, desktop)
- Deployed on Azure Static Web Apps
- Production-ready build using Vite

---

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- HTML5 & CSS3
- Git & GitHub
- Azure Static Web Apps
